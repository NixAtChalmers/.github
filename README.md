# Nix at Chalmers

A collection of resources for using Nix while studying at Chalmers.

**DISCLAIMER:** Nix at Chalmers is not associated with Chalmers University of Technology nor Chalmers Student Union.

## Our Goal

Our goal is to make using Nix for studying at Chalmers easier.

## What we do

- Package course software with Nix flakes. Check out our [repositiories](https://github.com/orgs/NixAtChalmers/repositories)!
- Curate a list of relevant resources at [NixAtChalmers/collection](https://github.com/NixAtChalmers/collection).
