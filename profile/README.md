# Linux at Chalmers

A collection of resources for using Linux while studying at Chalmers.

**DISCLAIMER:** Linux at Chalmers is not associated with Chalmers University of Technology nor Chalmers Student Union.

## Our Goal

Our goal is to make using Linux for studying at Chalmers easier.

## What we do

- Curate a list of relevant resources at [LinuxAtChalmers/collection](https://github.com/LinuxAtChalmers/collection).
- Package course software with Nix flakes. Check out our [repositiories](https://github.com/orgs/LinuxAtChalmers/repositories)!
